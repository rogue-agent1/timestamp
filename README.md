# timestamp

Convert between epoch, ISO 8601, and human-readable dates.

One file. Zero deps. Speaks time.

## Usage

```bash
python3 timestamp.py now
python3 timestamp.py 1741638000
python3 timestamp.py "2026-03-10T14:00:00"
python3 timestamp.py "March 10, 2026"
python3 timestamp.py diff 1741638000 1741724400
python3 timestamp.py add 1741638000 2h30m
```

## Requirements

Python 3.8+. No dependencies.
